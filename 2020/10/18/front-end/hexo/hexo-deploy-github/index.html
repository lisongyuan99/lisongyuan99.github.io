Hexo 一键部署 Github Pages

发表于 2020-10-18 分类于 前端 ， Hexo

步骤

  1. Github 新建 <用户名>.github.io 仓库

  2. 新建 gh-pages 分支

  3. settings->Github Pages 选择 gh-pages 分支

  4. 配置 -config.yml

  5. 一键部署

补充

使用 master dev 分支存储源文件
